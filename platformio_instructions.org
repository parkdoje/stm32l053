* Initialize a project

-  activate virtual env (=source env/bin/activate=)
-  =mkdir examplex=
-  =cd examplex && platformio init -b nucleo_l053r8=
-  edit =main.cpp=

* Running

-  =cd examplex= and =platformio run=
